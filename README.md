# odoo19-payroll-indonesia

Repository ini berisi addon payroll Indonesia untuk Odoo 19.

## Addon yang tersedia

- `l10n_id_hr_payroll`: fondasi awal payroll Indonesia untuk Odoo 19 Community
- `l10n_id_payroll_pph21`: scaffold ekstensi **Odoo 19 Enterprise Payroll** khusus **PPh21 Indonesia**

- [`l10n_id_hr_payroll/README.md`](l10n_id_hr_payroll/README.md)
- [`l10n_id_payroll_pph21/README.md`](l10n_id_payroll_pph21/README.md)
